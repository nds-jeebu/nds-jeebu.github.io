I was pretty vague about all of this last time and made sure to point out that we did not have the proper conditions to use the fixed point theorem. Here, I will make everything clear!

Define (Pseudo-Bellman Operator):

    Rmax∈RR_{max} \in \RRmax​∈R.
  2. Introduce a discount to our value functions:γ∈[0,1) \gamma \in [0,1)γ∈[0,1)
  3. To ensure the space is complete, we can restrict ourselves by assuming:

∣A∣<+∞ |A| < +\infin∣A∣<+∞

∣S∣<+∞|S| < +\infin ∣S∣<+∞

Updated Value Function

Putting these two together we have a new value function!

Define (Bounded, Discounted Value Function):
Vπ(s)=Eπ[∑t=0∞γtrt+1∣s0=s]V^{\pi}(s) = \mathbb{E_{\pi}}[\sum_{t=0}^{\infin}\gamma^tr_{t+1}|s_0=s]Vπ(s)=Eπ​[t=0∑∞​γtrt+1​∣s0​=s]

We can work through the same steps from post number 2 to get:Vπ(s)=∑s′∈S[P(s′∣s,πA(a∣s))(R(s,πA(a∣s),s′)+γVπ(s′))]V^{\pi}(s)= \sum_{s' \in S}[P(s'|s, \pi_A(a|s))(R(s,\pi_A(a|s),s')+\gamma V^{\pi}(s'))]Vπ(s)=s′∈S∑​[P(s′∣s,πA​(a∣s))(R(s,πA​(a∣s),s′)+γVπ(s′))]

Question: Okay, what’s different about this both mathematically and intuitively?
Answer:

  1. Mathematically: the value function is guaranteed to be bounded and finite! We will exactly how later in the post.
  2. Our agent now cares more about immediate rewards than future rewards.

Discussion (Discounting):

Personally, I stand by the fact that discounts are part of the formulation out of mathematical convenience. This isn’t really me taking shots at the formulation. Instead, this is me showing you where they really come from.

Since we kind of have to use a discount, you will see why in the next section, it is important to make sure that it intuitively makes sense. I want to reiterate:

Our agent now cares more about immediate rewards than future rewards.

Question: Why?
Answer: Think of what happens at different time steps:

  • at time t=0t=0t=0, γ0=1\gamma^0=1γ0=1, so the reward is the same.
  • at time t=1,γ1<1t=1, \gamma^1 <1t=1,γ1<1, so the reward is less than it would without the discount.
  • continue this and we get lim⁡t→∞γt=0\lim_{t \rightarrow \infin} \gamma^t=0limt→∞​γt=0, so distant future rewards are worth very little!

This is pretty reasonable because I can see how an agent doesn’t really care about what’s happening infinitely into the future.

Qπ(s,a)=∑s′∈S[P(s′∣s,a)(R(s,a,s′)+γVπ(s′))]Q^{\pi}(s,a)= \sum_{s' \in S}[P(s'|s, a)(R(s,a,s')+\gamma V^{\pi}(s'))]Qπ(s,a)=s′∈S∑​[P(s′∣s,a)(R(s,a,s′)+γVπ(s′))]

Define (Optimal Bounded, Discounted Quality Function):
Q∗(s,a)=∑s′∈S[P(s′∣s,a)(R(s,a,s′)+γV∗(s′))]Q^{*}(s,a)= \sum_{s' \in S}[P(s'|s, a)(R(s,a,s')+\gamma V^{*}(s'))]Q∗(s,a)=s′∈S∑​[P(s′∣s,a)(R(s,a,s′)+γV∗(s′))]

Completing the Proof:

Now, we are ready to get back to the proof.

First, let’s define the actual Bellman Operator:

Define (Bellman Operator):
Tπ:B(S)→B(S) T^{\pi}: B(S) \rightarrow B(S)Tπ:B(S)→B(S)

(TπV)(s)=∑s′∈S[P(s′∣s,πA(a∣s))(R(s,πA(a∣s),s′)+γVπ(s′))] (T^{\pi}V)(s)= \sum_{s' \in S}[P(s'|s, \pi_A(a|s))(R(s,\pi_A(a|s),s')+\gamma V^{\pi}(s'))](TπV)(s)=s′∈S∑​[P(s′∣s,πA​(a∣s))(R(s,πA​(a∣s),s′)+γVπ(s′))]

We can finally show that the Bellman Operator is a contraction as follows:

Take U,V∈B(S)U, V \in B(S)U,V∈B(S) and we get:
∣∣TπU−TπV∣∣∞=sup⁡s∈S∣∑s′∈S[P(s′∣s,πA(a∣s))(R(s,πA(a∣s),s′)+γU(s′))]−∑s′∈S[P(s′∣s,πA(a∣s))(R(s,πA(a∣s),s′)+γV(s′))]∣ ||T^{\pi}U-T^{\pi}V||_{\infin}= \sup_{s \in S}|\sum_{s' \in S}[P(s'|s, \pi_A(a|s))(R(s,\pi_A(a|s),s')+\gamma U(s'))] - \\ \sum_{s' \in S}[P(s'|s, \pi_A(a|s))(R(s,\pi_A(a|s),s')+\gamma V(s'))]|∣∣TπU−TπV∣∣∞​=s∈Ssup​∣s′∈S∑​[P(s′∣s,πA​(a∣s))(R(s,πA​(a∣s),s′)+γU(s′))]−s′∈S∑​[P(s′∣s,πA​(a∣s))(R(s,πA​(a∣s),s′)+γV(s′))]∣

=sup⁡s∈S∣∑s′∈S[P(s′∣s,πA(a∣s))γU(s′)]−∑s′∈S[P(s′∣s,πA(a∣s))γV(s′)]∣ = \sup_{s \in S}|\sum_{s' \in S}[P(s'|s, \pi_A(a|s))\gamma U(s')] - \sum_{s' \in S}[P(s'|s, \pi_A(a|s))\gamma V(s')]|=s∈Ssup​∣s′∈S∑​[P(s′∣s,πA​(a∣s))γU(s′)]−s′∈S∑​[P(s′∣s,πA​(a∣s))γV(s′)]∣

=γsup⁡s∈S∣∑s′∈S[P(s′∣s,πA(a∣s))(U(s′)−V(s′))]= \gamma \sup_{s \in S}|\sum_{s' \in S}[P(s'|s, \pi_A(a|s))(U(s')-V(s'))] =γs∈Ssup​∣s′∈S∑​[P(s′∣s,πA​(a∣s))(U(s′)−V(s′))]
Apply the triangle inequality:
≤γsup⁡s∈S∑s′∈SP(s′∣s,πA(a∣s))∣(U(s′)−V(s′))∣ \leq \gamma \sup_{s \in S}\sum_{s' \in S}P(s'|s, \pi_A(a|s))|(U(s')-V(s'))| ≤γs∈Ssup​s′∈S∑​P(s′∣s,πA​(a∣s))∣(U(s′)−V(s′))∣
Observe that ∣U(s′)−V(s′)∣≤∣∣U−V∣∣∞|U(s')-V(s')| \leq ||U-V||_{\infin}∣U(s′)−V(s′)∣≤∣∣U−V∣∣∞​ since the infinity norm is defined as the largest different in these two functions on their domain!
≤γsup⁡s∈S∑s′∈SP(s′∣s,πA(a∣s))∣∣U−V∣∣∞ \leq \gamma \sup_{s \in S}\sum_{s' \in S}P(s'|s, \pi_A(a|s))||U-V||_{\infin}≤γs∈Ssup​s′∈S∑​P(s′∣s,πA​(a∣s))∣∣U−V∣∣∞​
Now, there is no more dependence on s′s's′ so we get:
=γ∣∣U−V∣∣∞ = \gamma ||U-V||_{\infin}=γ∣∣U−V∣∣∞​

Let’s just rewrite this final result as:
∣∣TπU−TπV∣∣∞≤γ∣∣U−V∣∣∞||T^{\pi}U-T^{\pi}V||_{\infin} \leq \gamma ||U-V||_{\infin}∣∣TπU−TπV∣∣∞​≤γ∣∣U−V∣∣∞​
WOH! Recall how we defined γ∈[0,1)\gamma \in [0,1)γ∈[0,1), which is exactly what it needs to be to ensure that TπT^{\pi}Tπ is a contraction.

Banach’s Fixed Point Theorem:

Alright, it took a bit of grinding, but we finally have that the Bellman Operator is a contraction defined on a complete, normed space.

Recall the relationship:
(TπVπ)(s)=Vπ(s) (T^{\pi}V^{\pi})(s)=V^{\pi}(s)(TπVπ)(s)=Vπ(s)

The fixed point of the Bellman Operator is the Value function VπV^{\pi}Vπ !

We now define a procedure to compute Vπ(s)V^{\pi}(s)Vπ(s) as follows:

  1. Start with some an arbitrary V0V^0V0
  2. Apply TπT^\piTπ to V0V^0V0 until convergence!(V0,TπV0,TπTπV0,...) (V^0,T^{\pi}V^0,T^{\pi}T^{\pi}V^0,...)(V0,TπV0,TπTπV0,...)
  3. This sequence is guaranteed to converge to VπV^{\pi}Vπ!!!

Value Iteration (Finally):

Alright, so far we’ve figured out how to start with an arbitrary policy V0V^0V0 and have it converge to VπV^{\pi}Vπ. The problem is that VπV^{\pi}Vπ really isn’t guaranteeing that our agent is doing anything intelligible. What we really want is: an algorithm that starts with an arbitrary policy and converges to V∗V^*V∗.

Recall:
V∗(s)=maxaQ∗(s,a)V^*(s) = max_aQ^*(s,a) V∗(s)=maxa​Q∗(s,a)
Claim: We can use this relationship to create another operator with really powerful properties.

Here’s how (noting that sup⁡\supsup and maxmaxmax are quite analogous):

  1. Define (Bellman Optimality Operator):
    T∗:B(S)→B(S) T^{*}: B(S) \rightarrow B(S)T∗:B(S)→B(S)

(T∗V)(s)=sup⁡a∈A{∑s′∈S[P(s′∣s,a)(R(s,a,s′)+γV(s′))]} (T^{*}V)(s)= \sup_{a \in A}\{ \sum_{s' \in S}[P(s'|s, a)(R(s,a,s')+\gamma V(s'))]\}(T∗V)(s)=a∈Asup​{s′∈S∑​[P(s′∣s,a)(R(s,a,s′)+γV(s′))]}

  1. Claim: The Bellman Optimality Operator is a contraction on a complete normed space. (I recommend trying to prove this, the proof is very similar to the previous proof.)
  2. Observe (The Fixed Point): V∗(s)=(T∗V∗)(s)V^*(s)=(T^*V^*)(s)V∗(s)=(T∗V∗)(s)
  3. Apply the same procedure as the previous section. For arbitrary V0V^0V0 (V0,T∗V0,T∗T∗V0,...) (V^0, T^*V^0,T^*T^*V^0,...)(V0,T∗V0,T∗T∗V0,...)
    converges to V∗V^*V∗!!!

Discussion:

It helps me to rewrite the formula above in a way that parallels Newton’s Method. We are iteratively trying to get to a solution.

Define (Value Iteration):
Vk+1=T∗Vk,  k≥0 V^{k+1}=T^*V^k ,\ \ k \geq 0Vk+1=T∗Vk,  k≥0
